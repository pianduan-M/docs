## 概述

欢迎使用 TokenSmind 服务。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。我们深知个人信息对您的重要性，并会尽最大努力保护您的个人信息安全。使用我们的服务，即表示您同意本隐私政策中描述的数据处理程序。

## 公司信息

TokenSmind 由 **深圳市词源新智技术有限公司** 运营，我们的服务器和数据库部署于符合国际安全标准的云服务平台上，确保数据处理的安全性和可靠性。

## 信息收集

### 我们收集的信息

1. **账户信息**：当您注册 TokenSmind 账户时，我们会收集您提供的电子邮件地址、用户名和密码。
2. **使用数据**：我们会收集您使用我们服务的相关信息，包括：
   - API 调用记录
   - 模型使用情况
   - Token 消耗量
   - IP 地址和设备信息
   - 与模型请求相关的应用程序标识
3. **支付信息**：如果您购买我们的服务，我们会收集必要的支付信息，但完整的支付卡详情由我们的支付处理合作伙伴安全处理。
4. **企业信息**（如适用）：对于企业客户，我们可能收集公司名称、税号、发票信息等用于合规和账单目的的信息。

### 信息收集方式

1. **直接提供**：您在注册、使用或购买我们服务时直接提供的信息。
2. **自动收集**：通过 cookies、日志文件和其他技术自动收集的信息。
3. **第三方来源**：在某些情况下，我们可能从第三方服务提供商获取信息（例如身份验证服务）。

## 信息使用

我们使用收集的信息用于：

1. **提供服务**：处理 API 请求，提供大模型 (LLM) 访问服务。
2. **账户管理**：创建和管理您的账户，验证身份。
3. **计费和结算**：处理支付，发送账单，管理订阅，提供增值税发票（针对中国境内企业客户）。
4. **服务改进**：分析使用模式，优化我们的服务和 API 性能。
5. **客户支持**：回应您的问题和请求。
6. **安全保障**：检测和防止欺诈、滥用和安全威胁。
7. **通信**：发送服务通知、更新和营销信息（如您同意）。

## 隐私声明

我们高度重视用户隐私，致力于保护用户的个人信息。**我们不会收集、记录或存储用户通过 API 发送的任何提示内容（prompt）或模型返回的任何响应内容**。所有请求均通过端到端加密直接传输至模型提供商，不经我们的中间存储。我们不会向模型提供商透露您的身份信息（包括 IP 地址和用户代理字符串），除非法律强制要求。

## 信息共享

我们不会出售您的个人信息。我们可能在以下情况下共享您的信息：

1. **服务提供商**：与帮助我们提供服务的第三方服务提供商共享（如支付处理商、云服务提供商、分析服务）。
2. **合规要求**：当法律要求或为响应合法请求（如法院命令或传票）时。
3. **保护权利**：当我们认为披露是保护 TokenSmind、我们的用户或公众的权利和安全所必需的。
4. **业务转让**：如果 TokenSmind 参与合并、收购或资产出售，您的信息可能作为转让资产的一部分被转移。

这些第三方仅在履行其职能所需的必要情况下才能访问您的信息，不得将其用于其他目的。

## 数据安全

我们实施了适当的技术和组织措施来保护您的个人信息，包括：

1. **加密传输**：所有 API 通信通过 TLS/SSL 加密。
2. **安全存储**：敏感数据使用行业标准加密存储。
3. **访问控制**：严格限制员工对个人数据的访问。
4. **定期审计**：定期评估和更新我们的安全措施。

尽管我们努力保护您的个人信息，但请注意，没有任何安全措施是完全无懈可击的。

## 数据保留

我们会在实现本隐私政策中描述的目的所需的时间内保留您的个人信息，除非法律要求或允许更长的保留期。当不再需要您的个人信息时，我们将安全地删除或匿名化这些信息。

## 数据存储和国际传输

我们的服务器和数据库部署于符合国际安全标准的云服务平台上。如果您位于中国境外，请注意您的信息可能会被传输到中国并在那里处理。中国的数据保护法律可能与您所在国家/地区的法律不同。通过使用我们的服务，您同意将您的信息传输到中国并在那里处理。

## API 调用内容

1. **用户输入**：通过我们 API 发送到大模型 (LLM) 的提示和查询内容。
2. **模型输出**：大模型 (LLM) 生成的响应内容。

### 关于 API 调用内容的特别说明

- 我们不会将您通过 API 发送的内容用于训练我们自己的模型或任何第三方模型。
- 我们不会主动审查 API 调用内容。
- 模型提供商的数据政策由其各自决定。要了解特定模型提供商的数据处理实践，请查阅其官方文档。

## 您的权利和选择

您可以随时通过我们平台的用户界面更新可修改的个人信息。如果您希望请求删除您的账户和个人数据，请访问您的设置页面或通过下方联系信息与我们联系。

## Cookie 政策

我们使用 cookies 和类似技术来提供、保护和改进我们的服务。cookies 是存储在您设备上的小文本文件，用于记住您的偏好和其他信息。使用我们的服务即表示您同意我们使用这些方法。您可以通过浏览器设置控制和删除 cookies。但请注意，禁用 cookies 可能会影响我们网站的某些功能。

## 儿童隐私

我们的服务不面向 13 岁以下的儿童。我们不会故意收集 13 岁以下儿童的个人信息。如果您发现我们可能收集了 13 岁以下儿童的个人信息，请立即联系我们，我们将采取措施删除这些信息。

## 隐私政策更新

我们可能会不时更新本隐私政策，恕不另行通知。当我们进行重大更改时，我们会在网站上发布通知，并更新政策顶部的「最后更新」日期。在发布更改后，您继续使用我们的服务即表示您接受并同意这些更改。我们鼓励您定期查看本隐私政策，以了解我们如何保护您的信息。

## 适用法律

本隐私政策受中华人民共和国法律管辖。与本隐私政策相关的任何争议应提交给深圳市有管辖权的人民法院诉讼解决。

终止后，除本质上应继续存在的权利和义务外，所有权利和义务均将消失，包括但不限于所有权条款、免责声明、责任限制和争议解决条款。
